Set up your SSH key

ssh-to-go needs an SSH keypair to connect to your hosts. You can generate a new one or import an existing key.

Generate a new keypair

Creates a new ed25519 key. You'll need to add the public key to ~/.ssh/authorized_keys on each target host.
